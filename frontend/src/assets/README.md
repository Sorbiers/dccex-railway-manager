# Assets Folder

This folder contains static assets for the DCC-EX Control application.

## Required Files

### logo.png
- **Purpose**: Displayed in the toolbar/header
- **Recommended size**: 400x400 pixels or similar (will be scaled to 40px height)
- **Format**: PNG with transparent background recommended
- **Location**: `src/assets/logo.png`

### favicon.ico
- **Purpose**: Browser tab icon
- **Recommended size**: 32x32 or 16x16 pixels (multi-resolution ICO recommended)
- **Format**: ICO format
- **Location**: `src/assets/favicon.ico`

## Adding Your Files

1. Place your `logo.png` file in this directory
2. Place your `favicon.ico` file in this directory
3. Restart the development server if running

The application will automatically use these files once they are in place.
